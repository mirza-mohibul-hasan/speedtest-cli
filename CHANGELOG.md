# Changelog

## 1.1.0 - 2026-06-05

- Added `speedtest diagnose [target]` for network path diagnostics.
- Added DNS, TCP, TLS, HTTP reachability, request loss, latency, and jitter checks.
- Added `--json`, `--samples`, and `--timeout` options for diagnostics automation.
- Updated README and package metadata for the network diagnostics direction.

## 1.0.0 - 2026-06-04

- Initial release of the Node.js ESM speedtest CLI.
- Added ping, download, upload, result tables, JSON and CSV output, history, config, server selection, and watch mode.
- Added ESLint, Prettier, GitHub Actions CI, and npm release workflow.
